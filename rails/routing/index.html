Rails 3 routing


General notes

Routing should be done by modifying config/routes.rb file, where new routes are added. Example for static pages could be: 
      SampleApp::Application.routes.draw so
        match '/contact', :to => 'pages#contact'
        match '/about', :to => 'pages#about'
        match '/help', :to => 'pages#help'
        ...
      end
This, for example, matches /contact route to Pages controller, Contact action. To redirect route to '/' (or homepage), however, there is another method that should be used in routes.rb: 
      root :to => 'pages#home'
