Controller guidelines


Add static pages

All pages located in public folder can be accessible from the browser, without the need to add routes. To add static pages by using controller (generates Pages controller with actions home and contact): 
      > rails generate controller Pages home contact
This will update config/routes.rb file to add statements for getting these pages.

Adding pages manually

To add a page manually, you should first add it's action in the current controller class, for example: 
      class PagesController < ApplicationController
        ...
        def about 
        end
Then about action should be added into routes file (for example): 
      SampleApp::Aplication.routes.draw do
        ...
        get "pages/about"
        ...
      end
And finally, view should be added as well (in this case, you should create file about.html.erb with corresponding markup).
